# TECH-实验服管理系统-审批流程

## 1. 审批流程概述

本文档详细描述了“实验服管理系统”中涉及的各项业务审批流程，包括采购申请审批、领用申请审批和报废申请审批。每个流程都将明确审批节点、审批人、审批条件和流转规则，以确保业务的合规性和效率。

## 2. 采购申请审批流程

### 2.1. 流程图

```mermaid
graph TD
    A[发起采购申请] --> B{部门经理审批}
    B -- 批准 --> C{采购部审批}
    B -- 驳回 --> F[流程结束 (驳回)]
    C -- 批准 --> D{财务部审批}
    C -- 驳回 --> F
    D -- 批准 --> E[采购执行]
    D -- 驳回 --> F
    E --> G[入库验收]
    G --> H[更新库存]
    H --> I[记录入库日志]
    I --> F
```

### 2.2. 流程说明

1. **发起采购申请**：当实验服库存低于预警阈值时，系统自动触发或由仓库管理员手动发起采购申请。申请内容包括实验服型号、数量、预计到货日期等。
2. **部门经理审批**：
    * **审批人**：发起采购申请部门的经理。
    * **审批条件**：根据采购金额大小或采购数量，可设置不同审批等级。
    * **审批操作**：批准或驳回。
    * **流转**：批准后流转至采购部审批；驳回则流程结束。
3. **采购部审批**：
    * **审批人**：采购部负责人。
    * **审批条件**：对采购的合规性、供应商选择进行审核。
    * **审批操作**：批准或驳回。
    * **流转**：批准后流转至财务部审批；驳回则流程结束。
4. **财务部审批**：
    * **审批人**：财务部负责人。
    * **审批条件**：对采购预算、资金使用进行审核。
    * **审批操作**：批准或驳回。
    * **流转**：批准后进入采购执行阶段；驳回则流程结束。
5. **采购执行**：采购订单下达给供应商，等待物品到货。
6. **入库验收**：物品到货后，由仓库管理员进行清点、质量验收。
7. **更新库存**：验收通过后，系统自动更新实验服库存数量。
8. **记录入库日志**：系统自动记录入库详情，包括数量、时间、操作人等。

## 3. 领用申请审批流程

### 3.1. 流程图

```mermaid
graph TD
    A[实验人员发起领用申请] --> B{部门管理员审批}
    B -- 批准 --> C{仓库管理员审批}
    B -- 驳回 --> E[流程结束 (驳回)]
    C -- 批准 --> D[发放实验服并处理超额扣款]
    C -- 驳回 --> E
    D --> E
```

### 3.2. 流程说明

1. **实验人员发起领用申请**：实验人员通过系统提交实验服领用申请，选择所需实验服型号。系统自动限制每次领用数量为1件。
2. **部门管理员审批**：
    * **审批人**：实验人员所属部门的管理员。
    * **审批条件**：审核领用合理性。
    * **审批操作**：批准或驳回。
    * **流转**：批准后流转至仓库管理员审批；驳回则流程结束。
3. **仓库管理员审批**：
    * **审批人**：仓库管理员。
    * **审批条件**：
        * 库存充足性：检查是否有足够库存。
        * 部门额度校验：系统自动校验该实验人员所属部门的月度领用额度。
    * **审批操作**：批准或驳回。
    * **流转**：批准后进入发放阶段；驳回则流程结束。
4. **发放实验服并处理超额扣款**：
    * 仓库管理员根据审批结果发放实验服。
    * 如果领用超出部门月度额度，系统自动计算超额部分的半价金额，并生成扣款记录，以便财务在次月工资中扣除。
    * 系统更新实验服状态为“已领用”，并更新库存。

## 4. 报废申请审批流程

### 4.1. 流程图

```mermaid
graph TD
    A[发起报废申请] --> B{部门管理员审批}
    B -- 批准 --> C{仓库管理员审批}
    B -- 驳回 --> E[流程结束 (驳回)]
    C -- 批准 --> D[报废处理]
    C -- 驳回 --> E
    D --> E
```

### 4.2. 流程说明

1. **发起报废申请**：实验人员或管理员对达到使用寿命或损坏的实验服发起报废申请，填写实验服ID和报废原因。
2. **部门管理员审批**：
    * **审批人**：报废实验服所属部门的管理员。
    * **审批条件**：审核报废原因的合理性。
    * **审批操作**：批准或驳回。
    * **流转**：批准后流转至仓库管理员审批；驳回则流程结束。
3. **仓库管理员审批**：
    * **审批人**：仓库管理员。
    * **审批条件**：核实实验服状态、确认报废。
    * **审批操作**：批准或驳回。
    * **流转**：批准后进入报废处理阶段；驳回则流程结束。
4. **报废处理**：仓库管理员根据审批结果，将实验服标记为“已报废”，并从库存中移除，更新系统状态。
