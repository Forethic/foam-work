# 0002-selenium-页面前进后退

## 标签

## 内容

```python {.line-numbers}

# 1. 创建浏览器对象
browser = webdriver.Chrome()

# 2. 访问指定的URL
browser.get("https://www.baidu.com")
time.sleep(2)
browser.get("https://www.taobao.com")
time.sleep(2)

# 3. 回退
browser.back()
time.sleep(2)

# 4. 前进
browser.forward()
time.sleep(3)
```
