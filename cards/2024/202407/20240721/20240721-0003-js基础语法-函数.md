## 内容

keyword: `function`

> Javascript 大小写敏感
> arguments, 默认参数，实参个数可以不用等于形参个数，多余的个数可以通过 arguments获取

```Javascript {.line-numbers}
// 自执行函数，在function前添加布尔预算符，就变成了表达式，可以执行后面的代码
// 可以不写!, 但是前面的语句必须用; 区隔
!(function () {
    console.log("Hello Javascript")
})(); // 
```
