# 网段划分

| 网络类型              | 建议网段（示例） | 是否需要 VPN 互通 | 说明                         |
| --------------------- | ---------------- | ----------------- | ---------------------------- |
| 无线网络              | 192.168.10.0/24  | 否                | 员工无线接入，仅访问本地资源 |
| 有线网络 - 打印机     | 192.168.20.0/24  | 否                | 本地共享打印，无需跨 VPN     |
| 有线网络 - 监控网     | 192.168.30.0/24  | 否                | 本地监控，独立隔离           |
| 有线网络 - 设备外网   | 192.168.40.0/24  | 否                | 设备访问公网，不进 VPN       |
| 有线网络 - 设备内网   | 192.168.50.0/24  | 是                | 需要与对端设备内网互访       |
| 有线网络 - 用户终端网 | 192.168.60.0/24  | 是                | 员工终端需访问对端服务器     |
| 有线网络 - 服务器网   | 192.168.70.0/24  | 是                | 核心业务服务器，需对端访问   |
| 网关管理网段          | 192.168.254.0/24 | 否                | 仅本地管理网关，不暴露到 VPN |
