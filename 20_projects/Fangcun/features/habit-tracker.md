# 习惯养成 (Habit Tracker)

**目标：** 帮助您建立和保持积极的习惯，可视化您的进步，并通过正向反馈激励您持续成长。

**场景描述：**

1. **场景：每日习惯打卡 (移动端)**
    - **用户：** 晚上睡觉前，您完成了今天的阅读计划。
    - **交互：** 您打开“方寸”App，进入“习惯”模块。看到“阅读30分钟”这个习惯，轻轻一点，它就被标记为完成。App可能会播放一个简短的成功音效或显示一个激励动画。
    - **结果：** 您的阅读习惯被打卡，连续打卡天数增加。您感受到完成任务的满足感。

2. **场景：新增、修改与目标设定 (桌面端)**
    - **用户：** 您决定开始一个新的习惯，例如“每月阅读4本书”或“每周健身3次”，并希望设定具体的量化目标。
    - **交互：** 您打开“方寸”桌面App，进入“习惯”管理界面。点击“新建习惯”，输入习惯名称（例如：“阅读”），然后设定**可量化目标**（例如：“每月阅读4本书”）。您可以设置目标周期（每日、每周、每月），以及完成标准（例如：每次阅读30分钟，或每本书计1点）。同时，您可以设定提醒时间，并在Markdown编辑器中为这个习惯添加激励自己的话或学习资源链接。
    - **结果：** 新习惯被添加，并附带明确的量化目标。系统将根据您的打卡记录，自动追踪目标的完成进度，并在“习惯进度回顾”中提供反馈。

3. **场景：习惯进度回顾 (桌面端)**
    - **用户：** 每月初，您回顾上个月的习惯养成情况，看看哪些习惯需要改进。
    - **交互：** 您在“方寸”桌面App中查看习惯的统计视图。可视化图表清晰地展示了每个习惯的完成率、连续打卡天数、中断次数等。
    - **结果：** 您可以看到自己的成长轨迹，发现需要改进的地方，并可以根据数据调整习惯目标。

4. **场景：许愿金与现实奖励 (移动端/桌面端)**
    - **用户：** 您成功完成了某个习惯的打卡，系统根据习惯的难度和重要性，奖励您一定数量的“许愿金”。当许愿金积累到一定额度时，您可以选择兑换成现实奖励。
    - **交互：** App在习惯打卡成功后，显示获得的许愿金数量。在“许愿金”模块，您可以查看当前累计的许愿金，并设置兑换目标（例如：累计1000许愿金，兑换50元奖励）。当达成目标时，App提示您将指定金额（例如：从您的储蓄账户）转入一个特殊的“奖励账户”。
    - **结果：** 习惯养成获得了实际的物质激励，虚拟的努力与现实的奖励挂钩，形成强大的正向反馈，让习惯养成更有动力。

5. **场景：灵活跳过习惯 (移动端/桌面端)**
    - **用户：** 您因为出差或生病，无法完成今天的某个习惯，但又不希望因此中断连续打卡的记录。
    - **交互：** 在习惯打卡界面，您可以选择“跳过”今天的习惯。系统会提示您选择跳过原因，并记录下来。这次跳过不会中断您的连续打卡天数，但会影响当天的完成率。
    - **结果：** 习惯养成更加灵活，适应了生活的不可预测性，避免了因特殊情况而产生的挫败感，让您更容易坚持下去。

6. **场景：目标设定与追踪 (桌面端)**
    - **用户：** 您希望设定一个具体的、可量化的目标来驱动习惯养成，例如“每月阅读4本书”或“每周健身3次”。
    - **交互：** 在习惯管理界面，您可以为每个习惯设定目标（例如：数量、频率、持续时间）。App会根据您的打卡记录，自动追踪目标的完成进度，并在达成目标时给予反馈。
    - **结果：** 习惯养成不再是盲目的打卡，而是与明确的目标挂钩，让您的努力更有方向和成就感。

7. **场景：与日程安排联动 (移动端/桌面端)**
    - **用户：** 您希望在日程中看到需要打卡的习惯，并在打卡后自动更新日程状态。
    - **交互：** 在日程视图中，您可以选择将特定习惯显示为日程项。当您在习惯模块完成打卡时，日程视图中对应的习惯项会自动标记为完成。反之，如果在日程视图中标记习惯为完成，也会同步更新习惯模块的状态。
    - **结果：** 习惯和日程管理无缝衔接，在一个统一的界面下管理所有每日待办事项，提高效率并避免信息孤岛。
