# 软考高项-备考作战总纲 (V11.0 - 终极完整版)

## 核心战略

采用**“领先一步”的迭代学习法**，将备考完美融入日常。通过“**自学预习 -> 跟学深化 -> 碎片复习**”的三重循环，实现知识的螺旋式上升，确保“高项备考”与“FangCun开发”双线并行，最终高分通过。

## 时间承诺 (可持续且现实)

- **高项学习**: **工作日 1.5 - 2 小时** | **周末 3 - 4 小时**
- **FangCun开发**: **工作日 0.5 - 1 小时** | **周末 2 - 3 小时**
- **原则**: 追求每周总时长的稳定，而非每日的完美执行。保持节奏，允许休息。

---

### **第一阶段：自学预习，抢占先机 (今日 ~ 直播课开始)**

- **目标**: 启动学习，建立“身位领先”优势。
- **核心方法论**: **SQ3R法 + 两遍笔记法 (第一遍)**
- **详细安排**:
    1. **资料整理**:
        - [x] 已购买核心备考课程。
        - [x] 已下载《信息系统项目管理师教程（第4版）》PDF。
        - [x] 已在`20_projects/Sys_Integration_PM_Engineer/`目录下，创建`01_教材与大纲`, `02_历年真题`, `03_学习笔记`, `04_论文专题`等文件夹。
        - [ ] 在`04_论文专题/`下，创建`FangCun项目管理实践反思.md`核心文档。
    2. **执行“第一遍”学习**:
        - **周计划**: 每周设定攻克**1-2个章节**的目标。
        - **高效听课**: 以1.25-1.5倍速观看对应章节的录播课。
        - **“关键词/疑问”笔记**: 听课时，只快速记录核心关键词、疑问点，为第二遍精学做准备。
    3. **结构化积累论文素材**:
        - 每周开发`FangCun`后，花15分钟，在`FangCun项目管理实践反思.md`中，**按照“十大管理”框架**记录本周的管理实践与思考。

---

### **第二阶段：跟学深化，构建体系 (直播课期间 ~ 26年3月)**

- **目标**: **带着问题听课**，完成知识的第二遍精学，并同步攻克案例与论文。
- **核心方法论**: **翻转课堂 (Flipped Classroom) + 刻意练习**
- **详细安排 (每周循环)**:
    1. **第二遍：跟学深化 (利用直播回放)**
        - **课前**: 花15分钟回顾第一轮的“关键词”笔记，带着疑问听课。
        - **课中**: 从容地将“草稿”笔记，精炼、重构成体系化的正式笔记，存入`03_学习笔记/`。
    2. **课后巩固与检验**:
        - **小程序刷题**: 立即使用小程序题库，完成本周章节的**专项练习**（目标：50-100题），利用其即时反馈和错题归集功能。
        - **真题检验**: 在信管网完成至少一套本章的**案例分析题**。
    3. **月度回顾**:
        - **每月底的最后一个周末**，安排“月度回顾日”，暂停学习新知识，快速、完整地翻阅本月所有的笔记和错题集，将知识串联成网。

---

### **第三阶段：冲刺与内化 (26年4月 ~ 考前)**

- **目标**: 将所有知识融会贯通，达到“肌肉记忆”般的应试状态。
- **核心方法论**: **模拟考试 + 间隔重复**
- **详细安排**:
    1. **专题强攻**:
        - **论文**: 将`FangCun项目管理实践反思.md`中的素材，按照论文专题课的指导，系统性地写出8-10个不同主题的完整论文。
        - **案例**: 二刷、三刷近5-8年的所有案例分析真题。
    2. **全面复习与记忆**:
        - **音频复习法**: 将精炼后的笔记录制成音频，在通勤、散步等碎片化时间反复收听。
        - **间隔重复 (小程序主导)**: 每日利用碎片时间，在小程序上刷**“每日一练”**和**“错题集回顾”**。
    3. **全真模拟**:
        - 考前最后三周，安排**至少3次**完整的全真模拟考试，严格计时，提前适应考试节奏和压力。
