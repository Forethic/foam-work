## 📖 如何使用本知识库

### 对于管理层

1. 先看 [信息化蓝图](00_蓝图规划层/信息化蓝图.md) 了解3-5年愿景
2. 重点看 [分阶段选型策略](00_蓝图规划层/01_关键决策与选型/分阶段选型策略-超精益方案.md) 了解超精益实施方案（第一年8万投入）
3. 关注 [实施路线图](60_项目交付域/实施路线图-传统架构版.md) 了解传统架构参考

### 对于业务部门

1. 查看 [业务架构分析](10_业务架构域/业务架构分析.md) 了解业务需求
2. 了解 [应用架构设计](20_应用架构域/应用架构设计.md) 中相关系统设计
3. 参与系统实施评审

### 对于IT人员

1. 熟悉 [分阶段选型策略](00_蓝图规划层/01_关键决策与选型/分阶段选型策略-超精益方案.md) - 核心实施策略
2. 深入学习 [技术架构规划](30_技术架构域/技术架构规划.md)
3. 掌握宜搭开发能力，准备后续系统构建

### 快速开始

**新人必读文档（按顺序）**：

1. [信息化蓝图](00_蓝图规划层/信息化蓝图.md) - 了解整体愿景
2. [分阶段选型策略](00_蓝图规划层/01_关键决策与选型/分阶段选型策略-超精益方案.md) - **最核心**，理解超精益实施策略
3. [系统选型决策](00_蓝图规划层/01_关键决策与选型/系统选型决策-宜搭vs专业系统.md) - 理解选型逻辑
4. [项目交付计划](60_项目交付域/项目交付计划-传统架构版.md) - 了解项目组织（传统架构参考）
