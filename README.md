# 🧠 OLYD — AI 驱动的个人生活管理系统

基于 **VS Code + Foam + GitHub Copilot**，12 个自定义 Agent 覆盖从人生愿景到每日执行的完整闭环。

---

## 🤖 Agent 系统

| 层级     | Agent          | 用途                                               |
| :------- | :------------- | :------------------------------------------------- |
| **入口** | `olyd.day`     | 晨间仪表板 — 聚合任务、习惯、复习卡片、日程        |
|          | `olyd.inbox`   | 快速捕捉 — 零延迟记录想法                          |
| **规划** | `olyd.init`    | 战略初始化 — 人生愿景 → 全维度规约 (SoT)           |
|          | `olyd.year`    | 年度计划 — 季度里程碑 + 关键日期                   |
|          | `olyd.month`   | 月度计划 — 周度分解 + 固定日程                     |
|          | `olyd.week`    | 周排程 — 时间块 + 碎片转化                         |
| **执行** | `olyd.task`    | 任务卡片 — 生命周期管理 + ABC 优先级               |
|          | `olyd.project` | 项目管理 — 阶段 / 里程碑 / 健康报告                |
|          | `olyd.habit`   | 习惯养成 — 打卡 / streak / 积分兑换 (1000 分 = ¥1) |
|          | `olyd.note`    | 知识卡片 — SM-2 间隔复习                           |
|          | `olyd.study`   | 学习计划 — 阶段拆解 / 进度追踪                     |
| **反馈** | `olyd.review`  | 全周期复盘 — 日 / 周 / 月 / 年 + 核心原则评分      |

### 核心链路

```
olyd.init → olyd.year → olyd.month → olyd.week → olyd.task
                                                      ↓
olyd.inbox ──────────────────────────────────→ olyd.task
                                                      ↓
olyd.day ←── task + habit + note ────────────── olyd.review
```

## 📂 文件夹结构

| 文件夹      | 用途                                      | 管理                     |
| :---------- | :---------------------------------------- | :----------------------- |
| `meta/`     | 系统宪法 (核心原则、ABC 法、文件约定)     | 人工                     |
| `inbox/`    | 灵感捕捉缓冲区                            | `olyd.inbox`             |
| `specs/`    | 全维度战略规约 (SoT)                      | `olyd.init`              |
| `journal/`  | 年/月/周计划 + 日日志 + 复盘              | `year/month/week/review` |
| `tasks/`    | 任务看板 (`0_todo/` `1_doing/` `2_done/`) | `olyd.task`              |
| `projects/` | 项目记录                                  | `olyd.project`           |
| `areas/`    | 责任领域 (健康 / 家庭 / 职业)             | 人工                     |
| `habits/`   | 习惯卡片 + `_ledger.md` + `_rewards.md`   | `olyd.habit`             |
| `notes/`    | 知识卡片 (`cards/`) + 牌组索引 (`decks/`) | `olyd.note`              |
| `study/`    | 学习计划 + 会话日志                       | `olyd.study`             |
| `archive/`  | 归档区                                    | —                        |

## 🎯 优先级体系

| 优先级 | 维度        | 说明                              |
| :----- | :---------- | :-------------------------------- |
| 🏥 #1  | 健康        | 体重基线不可妥协，7–8 月减重冲刺  |
| 📋 #2  | 职业 / 考证 | 软件设计师 (今年) → 架构师 (明年) |
| 🏠 #3  | 家庭        | 女儿教育、家务、配偶陪伴          |
| ☕ #4  | 恢复        | 麻将 / 游戏 / 运动 (配额内)       |
| 🔍 #5  | 探索 / 使命 | 全平台记录软件 (构思期)、研究生   |

## 🚀 每日工作流

```
☀️ 早晨: olyd.day    → 查看今日总览 (任务 + 习惯 + 复习 + 日程)
📝 随时: olyd.inbox  → 快速记录想法
⚡ 执行: olyd.task    → 管理任务卡片，更新状态
🔥 习惯: olyd.habit  → 打卡 + 积分
📖 学习: olyd.note    → SM-2 卡片复习
🌙 晚间: olyd.review  → 评分 + 反思 + 工作日志
```
