# 命名约定

| 控件类型     | 前缀 | 示例                                |
| ------------ | ---- | ----------------------------------- |
| Form         | frm  | frmMain, frmLogin, frmSettings      |
| Button       | btn  | btnOK, btnCancel, btnSave           |
| Label        | lbl  | lblName, lblStatus, lblTitle        |
| TextBox      | txt  | txtUserName, txtPassword, txtEmail  |
| ComboBox     | cmb  | cmbCountry, cmbStatus, cmbType      |
| ListBox      | lst  | lstItems, lstUsers, lstFiles        |
| CheckBox     | chk  | chkEnabled, chkVisible, chkRemember |
| RadioButton  | rad  | radMale, radFemale, radOption1      |
| DataGridView | dgv  | dgvUsers, dgvOrders, dgvProducts    |
| PictureBox   | pic  | picLogo, picAvatar, picPreview      |
| Panel        | pnl  | pnlMain, pnlHeader, pnlFooter       |
| GroupBox     | grp  | grpSettings, grpInfo, grpOptions    |
| MenuStrip    | mnu  | mnuMain, mnuContext, mnuFile        |
| ToolStrip    | tls  | tlsMain, tlsStatus, tlsTools        |
| TabControl   | tab  | tabMain, tabSettings, tabDetails    |
| TreeView     | trv  | trvFolders, trvCategories, trvMenu  |
| ProgressBar  | prg  | prgLoading, prgProgress, prgStatus  |
