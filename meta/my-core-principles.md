# 我的核心原则与评分标准

这是指导我每日行为和反思的根本准则。

---

## 评分标准

> 做到10分，没做到0分。每日评分，每周回顾。

| 核心原则     | 10分标准 (做到)    | 0分现象 (没做到)           |
| ------------ | ------------------ | -------------------------- |
| **认真**     | 第一次就做好       | 马虎、投入不够、有缺陷     |
| **快**       | 当日事当日毕       | 拖延、无价值创造、浪费时间 |
| **坚守承诺** | 说到做到           | 虚假、不诚实               |
| **保证完成** | 100%的目标完成率   | 放弃或不能坚持             |
| **乐观**     | 积极正向思考       | 悲观失望、抱怨、指责       |
| **自信**     | 勇于接受挑战       | 怀疑、害怕、经常说“不可能” |
| **爱与奉献** | 主动帮助他人       | 自私、只考虑自己、索取     |
| **绝不找借口** | 承担责任，解决问题 | 找借口、找理由、我是受害者 |

## 回顾指南 (Review Guideline)

“每日评分”是数据采集，“定期回顾”才是产生洞察、驱动成长的关键。

### 每周回顾 (Weekly Review)

**时间**: 每周日晚上，作为“每周回顾与计划”的一部分。
**目的**: 发现本周的行为**模式**。

**流程**:

1. **浏览每日评分**: 快速浏览本周所有每日笔记中的评分表。
2. **识别高低分项**:
    - **哪个原则**本周得分普遍较高？为什么？我做了什么对的事情？如何保持？
    - **哪个原则**本周得分普遍较低？为什么？它主要发生在什么**情境**下（例如：处理重复工作时？面对新挑战时？）？
3. **制定下周改进点**: 针对得分最低的那个原则，制定一个**具体的、可执行的**改进微策略。例如：“下周，每当我接到一个重复性任务时，都必须使用检查清单来避免马虎。”

### 每月回顾 (Monthly Review)

**时间**: 每个月的最后一个周日。
**目的**: 发现长期的**趋势**和**根本原因**。

**流程**:

1. **回顾四周的“每周回顾”**: 查看过去四周，我都在试图改进哪些问题？效果如何？
2. **分析趋势**:
    - 某个原则的得分是在**持续上升**，还是在**反复波动**，或者**持续低迷**？
    - 持续低迷的原则，其**根本原因**可能是什么？（例如：“自信”持续低分，可能不是因为胆小，而是因为某项核心技能的缺失导致了不自信。）
3. **调整下月策略**: 根据对根本原因的判断，调整下个月的个人发展目标。可能不是“要更自信”，而是“花10个小时系统学习XX技能，以建立真正的自信”。
