# 任务: 邮箱搬家-域名解析切换

- **状态**: 完成
- **优先级**: A+
- **所属项目/领域**: [[Email_Procurement_Project/index]]
- **截止日期**: 2025-09-19
- **背景**: 这是切换邮件流向、正式启用新邮箱系统的关键一步。

---

## 目标 (Objective)

> 这个任务最终要达成的、可衡量的成果是什么？

- 将公司邮箱域名（如 `example.com`）的MX、SPF、DKIM等DNS记录，正确地指向新的网易邮箱服务器。

## 关键成果 (Key Results)

> 为了达成目标，需要交付哪些具体的成果？

- [x] 从网易邮箱获取所有需要配置的DNS记录值。
- [x] 登录公司域名的DNS服务商后台（如阿里云、腾讯云、Godaddy等）。
- [x] 在计划的切换时间点，修改MX、SPF、DKIM等记录。
- [x] 修改完成后，通过`ping`、`nslookup`或在线工具，验证DNS记录是否已生效。
- [x] 发送测试邮件，确认邮件收发已通过新系统。
